GitHub Settings

Your Personal Access Token authenticates uploads. It is stored only in your browser's local storage and never sent anywhere except the GitHub API.

Create one at GitHub → Settings → Developer Settings → Personal Access Tokens → Fine-grained tokens. Grant Contents: Read & Write on your repository.
